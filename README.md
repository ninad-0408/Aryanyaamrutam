## Aryanyaamrutam - A travel website

### Description
This is a website to book safaris in India Wildlife Sanctuaries and gives information about timings and location of sanctuaries.

### Technology used
- HTML 5
- CSS 3
- Bootstrap
- Jquery
- Nodejs
- Bootstrap-social
- Font-awesome
- Popper.js
